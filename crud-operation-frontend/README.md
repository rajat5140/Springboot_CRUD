This is the frontend of a CRUD application built using **React + Vite**.  
It connects to a **Spring Boot + MongoDB** backend using REST APIs.

## Features
- Create user
- View all users
- Get user by ID
- Update user
- Delete user

## Tech Stack
- React
- Vite
- JavaScript
- HTML & CSS

## Backend API
Base URL:
http://localhost:8081/api/users


## How to Run

### Prerequisites
- Node.js (v18+)
- Backend running on port 8081

### Steps

cd crud-operation-frontend

npm install

npm run dev

Frontend runs at:
http://localhost:5173
